# Katkı Sağlama Rehberi (CONTRIBUTING.md)

Bu projeye katkı sağladığınız için teşekkür ederiz! Lütfen aşağıdaki kuralları takip ediniz.

## 1. Geliştirme Ortamı
- Python 3.9+ kullanılması zorunludur.
- Tüm kodlar `src/` klasörü altında ilgili modülde yer almalıdır.

## 2. Commit Mesajları
Commit mesajları profesyonel ve açıklayıcı olmalıdır (Örn: `feat: add sar data fusion module`).

## 3. Kod Standartları
- Değişken ve fonksiyon isimleri İngilizce olmalıdır.
- Kullanıcıya yönelik loglar ve README belgeleri Türkçe/İngilizce çift dilli olabilir.

## 4. Pull Request (PR) Süreci
- Yeni bir özellik eklemeden önce mutlaka bir "Issue" açınız.
- PR'lar `main` branchine yapılmalı ve mutlaka bir `walkthrough.md` güncellemesi içermelidir.

*Birlikte hayat kurtaran teknolojiler geliştirebiliriz.*
